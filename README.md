# 🍽️ Zomato Data Analysis Dashboard (Power BI)

## 📌 Overview

This Power BI dashboard provides an interactive analysis of restaurant data sourced from Zomato. It offers insights into restaurant distribution, cuisines, ratings, votes, and service availability across multiple countries and cities.

The goal of this dashboard is to help users explore trends in the food industry, identify popular cuisines, and understand customer preferences based on ratings and votes.

---

## 📊 Key Features

### 🔢 Summary Metrics

* **Total Restaurants:** 9,551
* **Total Cuisines:** 1,826
* **Countries Covered:** 15
* **Cities Covered:** 141
* **Total Votes:** 1 Million
* **Average Rating:** 2.91

---

### 🌍 Filters

* **Country Selector:** Filter data by country (e.g., India, Australia, Brazil, etc.)
* **Year Selector:** Allows dynamic filtering of data by year

---

### 📈 Visualizations

#### 1. City-wise Restaurant Count

* Displays the number of restaurants in each city
* Highlights top cities such as New Delhi, Gurgaon, and Noida

#### 2. Cuisine Count vs Average Rating

* Combines:

  * Bar chart → Number of restaurants per cuisine
  * Line chart → Average rating per cuisine
* Helps identify both popularity and quality of cuisines

#### 3. Cuisine-wise Restaurant Distribution

* Shows the most common cuisines
* Examples:

  * North Indian
  * Chinese
  * Fast Food
  * Cafe & Bakery

#### 4. Table Booking Availability

* Donut chart showing:

  * Restaurants offering table booking
  * Restaurants without table booking

#### 5. Online Delivery Availability

* Donut chart showing:

  * Restaurants offering online delivery
  * Restaurants without delivery services

## 🚀 How to Use

1. Open the Power BI dashboard file (.pbix)
2. Use the filters (Country, Year) to customize your view
3. Hover over visuals for detailed insights
4. Analyze trends across cities, cuisines, and services
